# Stella

AI coding agent for your terminal.

## Install (Windows)

Open **PowerShell** and run:

```powershell
irm https://github.com/a1x10/stella/releases/latest/download/install.ps1 | iex
```

This downloads Stella, adds it to your PATH, and launches it. After that, just type:

```
stella
```

from any terminal, in any project folder.

## What it does

- Runs entirely in your terminal (TUI) and includes a built-in web interface (`stella web`).
- No Node, no Bun, no setup — a single self-contained executable.
