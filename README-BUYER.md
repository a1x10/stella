# Stella Coder - Инструкция для покупателя

## Установка — одна команда

Откройте **PowerShell** и вставьте одну строку:

```powershell
irm https://raw.githubusercontent.com/a1x10/stella/master/install.ps1 | iex
```

Готовая сборка скачивается напрямую с GitHub — **никакого `npm`, логинов и прав администратора не нужно**. Устанавливается всё автоматически:
- Node.js (если его нет) — портативно, в профиль пользователя
- сама Stella Coder (один файл ~2 МБ)
- команда `stella` прописывается во все терминалы

После этого `stella` **работает из любого терминала** — PowerShell, CMD, Git Bash, Windows Terminal. Если терминал был уже открыт — перезапустите его.

> Всё ставится в ваш профиль (`%LOCALAPPDATA%\StellaCoder`). Права администратора не нужны.

### Обновление

Просто запустите ту же команду ещё раз — она скачает свежую версию.

## Запуск

```
stella
```

## Настройка Telegram бота

1. В терминале Stella выполните:
```
/tg
```

2. Бот запустится. Теперь сгенерируйте код:
```
/tg-code
```

3. Вы получите 4-значный код (например: `5291`)

4. Дайте этот код своим друзьям/клиентам

## Как друзья могут управлять вашим компьютером

1. Заходят в Telegram
2. Ищут бота: @stella_CLI_bot
3. Нажимают `/start`
4. Вводят ваш код
5. Готово! Теперь они могут управлять вашим компьютером

## Примеры команд для друзей

- "выключи телевизор"
- "сделай скриншот"
- "открой браузер"
- "покажи систему"
- "выполни: dir"

## Управление

| Команда | Описание |
|---------|----------|
| `/tg` | Запустить бота |
| `/tg-code` | Сгенерировать код |
| `/tg-users` | Кто подключён |
| `/tg-stop` | Остановить бота |

## Безопасность

- Код привязан к вашему компьютеру
- Только люди с кодом могут подключиться
- Вы можете отключить бота: `/tg-stop`
бот работет по системе codex и человек не несет отвтствиность за точто длелает  или полльзует человек
